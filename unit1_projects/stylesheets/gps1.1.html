Why Pair Programming?

Pair programming is a useful concept in that it helps programmers solve coding problems. It allows you to let someone else look at your work, bounce ideas, and check for errors as they code. In addition, it also provides soft skill benefits as it allows each coder to learn from each others' styles, attitudes, problem solving thought process, and technical abilities. It also trains each person in their communication skills. Difference between driver and navigator

Drivers: Takes instruction from the navigator and writes the code down. For example, if the navigator says that we should change the color of a page header to red, the driver would add that stylistic change to the CSS. Navigator:- Navigator is somebody who basically throws ideas about how to proceed. For example, if two people are trying to design a html page, and are about write css for the html, navigator is somebody who would give driver idea like, making font 18px, or making image circular...
